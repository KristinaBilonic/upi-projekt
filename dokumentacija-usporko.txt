	DVA TERMINALA OBAVEZNO (prvo za 	[-npm run dev]	i za 	[node server.js])

	cd projekt
	npm install
	npm run dev

	Drugi terminal:
	za pokrenit server i spojit se na bazu:  cd projekt
											 cd backend
											 node server.js
